# SIEM RUST
SIEM built in Rust

## About
- Exploring how SIEMs work and trying to dive deeper into Rust. Research on SIEM are largely based off Splunk's univeral forwarder and indexer.


## Starting the Indexer
- cd siem-indexer
- cargo build
- cargo  run

## Starting the Forwarder
- cd siem-forwarder
- cargo build
- cargo run
